Introducción

Comience con Flex Design, el framework CSS con capacidad responsiva incorporada de código abierto para diseño de sitios web incorporando extensiones de javascript y edición de CSS.

Globales Importantes

Flex Design emplea múltiples estilos y configuraciones globales importantes que deberá tener en cuenta al usarlo, los cuales están orientados casi exclusivamente a la normalización de estilos de navegador cruzado.

HTML5 doctype

Flex Design requiere el uso del doctype HTML5. Sin él, verás un estilo incompleto, el incluirlo no debería causar ningún contratiempo.

<!doctype html>
<html lang="en">
...
</html>

Metaetiqueta responsiva

Para garantizar una reproducción adecuada y un zoom táctil para todos los dispositivos, agregue la metaetiqueta <meta> a su <head>. en todas sus páginas web:

<meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

Font Awesome

Font Awesome te ofrece iconos vectoriales escalables que se pueden personalizar al instante: tamaño, color, sombra paralela y cualquier cosa que se pueda hacer con el poder de CSS.

Descargar Font Awesome

Diseño

Flexdesign cuenta con un diseño enfocado al uso interactivo con interfaces amigables y responsivas. Con una paleta de colores minimalistas y combinables entre sí. Su diseño se adapta a las distintas resoluciones comerciales en el mercado, pc, Tablet y smartphone, brindando una vista agradable y de fácil interacción, por lo que resulta una navegación intuitiva y accesible

Contenedores

Los contenedores son el elemento de diseño más básico en Flex Design y son necesarios al usar nuestro sistema. Elija entre un contenedor responsvo de ancho de fluido (lo que significa que es 100% ancho todo el tiempo). La mayoría de los diseños requieren un contenedor y para un mejor diseño recomendamos anidar contenedores.

parallax

Te explicamos qué es el Parallax, ese efecto que al entrar en una web, hacer scroll el fondo parece moverse a una velocidad distinta que el contenido.
Flex Design implementa parallax como un contenedor que posiciona una imagen de fondo, sin afectar al contenido.

Tipografía

Utilizamos por defecto la funete fontin sans, la cual nos otorga una fuente con un diseño mas visual, respetando una buena lectura y letras sin diseños exagerados, y principalmente legible

Fuentes

//La ruta de las fuentes debe estar en la misma ubicacion que el flexdesign.css
//Puede usar una ruta alternativa editando el url
