Flex Design

Flex Desing

Fácil, rápido y flexible.
Flex Design, el framework CSS con capacidad responsiva incorporada de código abierto para diseño de sitios web incorporando extensiones de javascript y edición de CSS.

Versión actual v1.0.0

Instalación

Copia y pega en tu encabezado <head> para cargar nuestro CSS.

<!-- jQuery -->
<script src="../js/jquery-3.2.1.min.js"></script>
<!-- FlexDesign -->
<link rel="stylesheet" type="text/css" href="css/flexdesign.css">
<script src="js/flexdesign.js"></script>
<!-- Font Awesome -->
<link rel="stylesheet" type="text/css" href="css/font-awesome-4.7.0/css/font-awesome.min.css">

Requerimientos

Flex Design emplea múltiples estilos y configuraciones globales importantes que deberá tener en cuenta al usarlo, los cuales están orientados casi exclusivamente a la normalización de estilos de navegador cruzado.
